## Reporting a Vulnerability

We take the security of our project seriously. If you discover a security vulnerability, please follow these steps to report it:

1.  **Do not create a public issue.** Publicly disclosing a vulnerability can put the community at risk.
2.  **Email us.** Please send an email to [sapienzafoilingteam@gmail.com](mailto:sapienzafoilingteam@gmail.com) with a description of the vulnerability.
3.  **Include details.** Please provide as much information as possible, including:
    *   The type of vulnerability
    *   Steps to reproduce
    *   Potential impact
4.  **Wait for a response.** We will acknowledge your report within 48 hours and provide an estimated timeline for a fix.

Thank you for helping keep our project safe!
